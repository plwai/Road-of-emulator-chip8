# OglSP
A simple opengl utilities stater pack which include utilities like shader loader, texture loader, sprite rendering and etc. A simple example is included in this project. Currently, only 2D is supported.

# Dependency
- CMake 3.10+
- OpenGL 3.3+
- GLFW 3+
- GLM
- FreeType
- libogg
- ligVorbisFile
- OpenAL
